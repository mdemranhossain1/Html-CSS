This is the structure of an HTML document

HTML document always starts with <Html> tag and ends with </Html> tag.

Inside the <Html> tag, there are two main sections: <Head> and <Body>.

The <Head> section contains meta-information about the document, such as its title, character set, styles, and scripts. It is enclosed within <Head> and </Head> tags.

The <Body> section contains the actual content of the webpage that is visible to users. This includes text, images, links, and other elements. It is enclosed within <Body> and </Body> tags.
